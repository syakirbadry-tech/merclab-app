## 🚀 Deploy to GitHub Pages (2 Methods)

### Method 1: Automated (Recommended)
```bash
chmod +x deploy.sh
./deploy.sh
```
Then answer the prompts. Done! ✨

### Method 2: Manual
1. Go to **github.com** → Create new repo: `merclab-app`
2. Upload files via GitHub web interface
3. Settings → Pages → Enable GitHub Pages
4. Get your URL: `https://YOUR-USERNAME.github.io/merclab-app`

## 📱 Install on iPhone

1. Open Safari → Go to your GitHub Pages URL
2. Tap Share (↗️)
3. Tap **Add to Home Screen**
4. Tap **Add**

Done! Your app is now on your home screen. 🎉

## 🔄 How It Works

- **index.html** - Your app with PWA meta tags
- **manifest.json** - Tells devices how to display your app
- **service-worker.js** - Enables offline functionality
- **icons/** - Professional icons for all devices

The Service Worker caches everything, so your app works offline and loads faster after the first visit.
